# Resume Screening System

## Project Description
This project uses machine learning to automatically classify resumes
into different job categories.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression

## How It Works
1. Load resume dataset
2. Clean resume text
3. Convert text to numerical features using TF-IDF
4. Train machine learning model
5. Predict job category for new resumes

## Accuracy
Model accuracy: ~66%
